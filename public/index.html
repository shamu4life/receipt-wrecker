Receipt Wrecker

Your words as giant type, or a picture as glyph-art — paste the result into Twitch chat as a Cheer100 and it prints on the streamer’s receipt printer.

Big Text = your words printed BIG as real type (press Enter for line breaks). Image = a picture rendered as glyph-art. Both ride out on a 100-bit cheer.
Which characters build the picture. ASCII — letters renders in the printer's own font (no tofu) and is space+letters only, so it slips past symbol filters; with dither + run-capping it also dodges Nightbot's repetition filter — the safe default. Full detail adds punctuation for more contrast (may trip a symbol filter). Hanzi tiles from Han glyphs (letters, no markup — prints on the RP332). Blocks/Braille look crispest on screen but this printer usually can't render them — they print as blank boxes.
How many characters wide the picture is — more = finer detail but a longer message (watch the char count). ~20 fits comfortably; tall/portrait images may need fewer.

Big type prints horizontally as real type. Giant sideways runs it down the tape (each letter as tall as the paper is wide); line breaks become side-by-side columns. Hanzi tiling builds those same giant sideways letters from Han glyphs instead of CSS — no markup, so it slips past the symbol filters that block the HTML sideways payload, and it's the one style proven to print on this RP332. Set Detail (columns) to the printer's true width (Print test strip has a ruler); if it prints blank, the printer has no CJK font.
How big the letters are. For Big type it's the pixel font size (~72 fills the width with a short word; lower it for longer messages). For Hanzi tiling it's how much of the tape width the letters fill. Press Enter for separate lines.

Paste into the official Twitch web/mobile chat and send as a ≥100-bit cheer — only the first-party client actually cheers.

Held or blocked by chat? Bot filters (symbols, repetition, duplicates) move around as mods tune them. This tool already stacks the odds in your favor: a rotating nonce beats duplicate-blocking, letters-only art keeps the symbol ratio low, and run-capping avoids repetition walls. Short of a VIP or bot permit (which bypass all of it) there’s no way to guarantee delivery — so if a message is held, just resend, shorten it, or drop the column count and try again.
