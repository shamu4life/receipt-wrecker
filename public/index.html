Receipt Wrecker

Your words as giant type, or a picture (a real image by link, or glyph-art from a file) — paste the result into Twitch chat as a Cheer100 and it prints on the streamer’s receipt printer.

Big Text = your words printed BIG as real type (press Enter for line breaks). Image = the real picture from a link, or glyph-art from an uploaded file. Both ride out on a 100-bit cheer.

Add text and pictures; they print in order down the receipt. Reorder with ↑/↓, remove with ×. Too much for one receipt just tapes into more — each is its own cheer (see the bits total under the preview).
Which characters build the picture. ASCII — letters renders in the printer's own font (no tofu) and is space+letters only, so it slips past symbol filters; with dither + run-capping it also dodges Nightbot's repetition filter — the safe default. Full detail adds punctuation for more contrast (may trip a symbol filter). Hanzi tiles from Han glyphs (letters, no markup — prints on the RP332). Blocks/Braille look crispest on screen but this printer usually can't render them — they print as blank boxes.
How many characters wide the picture is — more = finer detail. The picture no longer has to fit one message: it splits into as many receipt strips as it needs, so higher detail just means more strips to tape together (in order). ~20 is a quick single strip; crank it up for a real picture.

Big type prints horizontally as real type. Giant sideways runs it down the tape (each letter as tall as the paper is wide); line breaks become side-by-side columns. Hanzi tiling builds those same giant sideways letters from Han glyphs instead of CSS — here each line stacks full-size down the tape (so every line keeps full detail) — no markup, so it slips past the symbol filters that block the HTML sideways payload, and it's the one style proven to print on this RP332. Set Detail (columns) to the printer's true width (Print test strip has a ruler); if it prints blank, the printer has no CJK font.
How big the text is. 100% = as large as fits the paper width (auto-shrinks so it never runs off the edge); drop it to leave margins. For Giant sideways this sets the letter size — smaller letters fit more text before each strip ends. Press Enter for separate lines.
Where each strip ends so it doesn’t cross the printer’s page break (which slices a word). This is a cap, not a target — slide it up until a strip just fills one physical receipt, then back off a touch. Raising it past where your printer actually cuts makes the gouging worse, so calibrate to your printer. Long messages split into more strips; tape them together in order.
Paste a link to an image already on the web and it prints the actual picture (far better than glyph-art). Memes are already hosted; for your own files, drop them in Discord/imgur first and paste the direct link — this tool can’t host a local file (no server). Leave blank to make glyph-art from an upload instead.
How wide it prints. 70 mm is the full printable width (an 80 mm roll with ~5 mm unprintable margins each side); the printer converts it to black & white, so high-contrast images (memes, logos, line art) come out cleanest.

Dithering a linked picture fetches it through this site’s server — your browser won’t let us read a remote image’s pixels any other way. Big Text and glyph-art still never leave your device.

Paste into the official Twitch web/mobile chat and send as a ≥100-bit cheer — only the first-party client actually cheers.

Held or blocked by chat? Bot filters (symbols, repetition, duplicates) move around as mods tune them. This tool already stacks the odds in your favor: a rotating nonce beats duplicate-blocking, letters-only art keeps the symbol ratio low, and run-capping avoids repetition walls. Short of a VIP or bot permit (which bypass all of it) there’s no way to guarantee delivery — so if a message is held, just resend, shorten it, or drop the column count and try again.
