Turns big text or a picture into paste-ready block glyphs — paste the result into Twitch chat as a Cheer100.

Big Text = giant vertical letters — turn the receipt to read. Add line breaks to split the width into side-by-side columns (packs more onto one receipt). Image = a dithered picture.
CJK ramp (Big-Text default) & Braille (Image default) both clear chat filters in practice — a bot may delete them after, but the print already fired on the cheer, so it still prints. One line = one big column down the tape (turn the receipt to read). Press Enter to add another line — each line takes an equal slice of the width as its own column, so 2 lines = two half-width columns. Drag Size to scale the letters within each column live — lower = shorter strip / fewer receipts, higher = fills the column width. (Line breaks make columns; Size scales within them.)

Paste into the official Twitch web/mobile chat and send as a ≥100-bit cheer — only the first-party client actually cheers.

Held or blocked by chat? A wall of block symbols trips chat-bot symbol filters (e.g. Moobot’s “block symbols”, on by default) — the CJK tier uses letters and slips past most of them. The reliable fix: have the streamer make you a VIP (VIPs bypass AutoMod and every major bot’s filters), or add a bot permit / disable its “symbols” filter.
